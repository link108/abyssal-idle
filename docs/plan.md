# Abyssal Idle - High-Level Plan

Use this checklist as a living plan. Check items as they ship.

## Milestone 1: Core Loop (Playable)
- [ ] Fishing minigame (timing bar) with manual catch.
- [ ] Tin creation flow (process + add ingredients).
- [ ] Basic market sell mode toggle (fish vs tins).
- [ ] Upgrade list from JSON (buy, cost scaling, levels).
- [ ] Save/load (persist money, inventory, upgrades).

## Milestone 2: Automation Tier 1-2
- [ ] Add crew trips on fishing screen (send button, return timer).
- [ ] Crew upgrades: faster trips, bigger catches.
- [ ] Optional: multiple crew slots.
- [ ] Optional: crew hiring with stats/specialties.
- [ ] Crew speed upgrades.
- [ ] Crew quality upgrades (better average outcome).

## Milestone 3: Cannery + Processing
- [ ] Processing choices: smoke / ferment / age / skin.
- [ ] Auto-processing stations (single method per station).
- [ ] Ingredient inventory (buy + consume).

## Milestone 4: Sustainability & Tone
- [ ] Ocean health system (visible meter).
- [ ] Fishing pressure scales with automation.
- [ ] Narrative logs (cozy -> unsettling text).

## Milestone 5: Runs / Expeditions (Optional)
- [ ] Short expedition choices (risk/reward).
- [ ] Rare resources + story beats.

## Milestone 6: Endings
- [ ] Overfishing ending.
- [ ] Sustainable ending.
- [ ] Credits + summary screen.

## Milestone 7: Polish
- [ ] UI pass (cards, spacing, visual identity).
- [ ] Audio + sfx.
- [ ] Steam demo build.
